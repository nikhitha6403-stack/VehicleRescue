About
VehicleRescue

A real-world problem. A real solution. Built from scratch.

💡 The Problem This Solves

Every day, thousands of people in India break down on highways, remote roads, and city streets. They don't know what went wrong, and they have no idea where the nearest garage is. This is especially hard for bike riders who have almost no digital support tools.

🔧 What VehicleRescue Does

  • Detects vehicle faults and explains them in simple plain language
  • Works for both cars and bikes/scooters — unlike most existing apps
  • Uses GPS to find the 3 nearest garages in real time
  • Sends a one-tap SOS alert with your location + fault details to a garage
  • Tracks your vehicle's health history over time

🛠️ Tech Stack Used

HTML5 CSS3 JavaScript Geolocation API Google Maps API OBD2 / ELM327 Web Bluetooth API LocalStorage

🗺️ Project Roadmap

  • ✅ Phase 1 – UI prototype with manual fault selection
  • ✅ Phase 2 – GPS-based garage finder
  • 🔄 Phase 3 – Real OBD2 Bluetooth integration (in progress)
  • ⏳ Phase 4 – SMS/WhatsApp alert system via Twilio
  • ⏳ Phase 5 – Mobile app (React Native)

👨‍💻 Built By

This project was built as an open-source solution to a real problem faced by millions of vehicle owners in India. It is intended to grow into a full startup product with garage partnerships.
